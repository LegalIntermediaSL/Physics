# Sistemas Unidimensionales

Resolver la ecuación de Schrödinger en 1D para potenciales simples ilustra los efectos cuánticos más sorprendentes.

## Pozo de Potencial Infinito (Partícula en una Caja)
Un electrón confinado entre $x=0$ y $x=L$ con paredes impenetrables ($V=\infty$).
- **Energías Cuantizadas**:
$ E_n = \frac{n^2 \pi^2 \hbar^2}{2mL^2} $ para $n = 1, 2, 3, ...$
- **Funciones de onda**:
$ \psi_n(x) = \sqrt{\frac{2}{L}} \sin\left(\frac{n\pi x}{L}\right) $

## Oscilador Armónico Cuántico
Potencial cuadrático $V(x) = \frac{1}{2}m\omega^2 x^2$. Fundamental para vibraciones en moléculas y cristales.
- **Energías Cuantizadas**:
$ E_n = \hbar\omega \left(n + \frac{1}{2}\right) $ para $n = 0, 1, 2, ...$
- Notar que el estado de menor energía ($n=0$) no tiene energía cero: $E_0 = \frac{1}{2}\hbar\omega$ (Energía del punto cero).

## El Efecto Túnel
Si una partícula se acerca a una barrera de potencial de altura $V_0$ con una energía $E < V_0$, clásicamente rebotará siempre. Cuánticamente, la función de onda decae exponencialmente dentro de la barrera, pero si la barrera es delgada, existe una probabilidad no nula de que la partícula aparezca al otro lado.
