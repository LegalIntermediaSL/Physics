# Física Computacional

La física computacional convierte modelos teóricos en experimentos numéricos. Permite explorar sistemas demasiado complejos para resolverse a mano, visualizar soluciones, analizar datos reales y desarrollar intuición sobre la dinámica de modelos físicos.

## Temas de Estudio

- **Python científico**: `NumPy`, `SciPy`, `Matplotlib`, `Jupyter`.
- **Métodos numéricos básicos**: Integración, derivación, interpolación y búsqueda de raíces.
- **Ecuaciones diferenciales**: Integradores para EDOs y una primera aproximación a PDEs.
- **Ajuste y análisis de datos**: Regresión, barras de error, histogramas y pruebas estadísticas.
- **Transformadas y espectros**: FFT, análisis de frecuencia y convolución.
- **Simulación**: Monte Carlo, sistemas dinámicos, partículas, redes y modelos discretos.
- **Reproducibilidad**: Organización de notebooks, scripts, control de versiones y documentación mínima.

## Habilidades Clave

### 1. Pensamiento algorítmico
No basta con saber física; hay que traducir una ley o ecuación a pasos computables.

### 2. Control de errores
Toda simulación tiene errores de truncamiento, redondeo, discretización o muestreo.

### 3. Visualización
Graficar bien una trayectoria, un potencial o un espectro muchas veces revela la física mejor que una página de ecuaciones.
