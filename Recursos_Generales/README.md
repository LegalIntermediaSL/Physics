# Recursos Generales y Matemáticas

El lenguaje de la física es la matemática. Además, en esta era digital, existen simulaciones y canales de divulgación de altísima calidad que facilitan la comprensión intuitiva de conceptos complejos.

## Matemáticas para la Física

Para avanzar en el estudio de la física, se requieren bases sólidas en las siguientes áreas de las matemáticas:

1. **Cálculo (Diferencial e Integral)**: Fundamental para casi toda la física (cinemática, trabajo, áreas bajo la curva).
2. **Álgebra Lineal**: Matrices y vectores. Esencial para la mecánica cuántica y clásica multivariable.
3. **Cálculo Vectorial**: Gradiente, divergencia, rotacional. Indispensable para el electromagnetismo (Ecuaciones de Maxwell).
4. **Ecuaciones Diferenciales**: Para entender osciladores, circuitos, ondas y la ecuación de Schrödinger.

**Cursos de Matemáticas**:
- [Khan Academy - Matemáticas Superiores](https://es.khanacademy.org/math)
- [MIT 18.01, 18.02, 18.03](https://ocw.mit.edu/search/?d=Mathematics): Cursos de cálculo de una y varias variables, y ecuaciones diferenciales.

## Simuladores de Física

- **[PhET Interactive Simulations](https://phet.colorado.edu/es/)**: Simulaciones interactivas creadas por la Universidad de Colorado Boulder. Excelentes para visualizar conceptos abstractos.
- **[Algodoo](http://www.algodoo.com/)**: Un software tipo "sandbox" para experimentar con la mecánica clásica en 2D.
