# Repositorio de Física

¡Bienvenido al repositorio de aprendizaje de Física! Este espacio está diseñado para recopilar, estructurar y facilitar el estudio de las diversas ramas de la física. Aquí encontrarás rutas de aprendizaje, temas clave, y enlaces a cursos y artículos recomendados de alta calidad.

## Estructura del Repositorio

El repositorio está dividido en las siguientes áreas principales. Selecciona una para explorar sus temas y recursos:

- 🍎 **[Mecánica Clásica](./Mecanica_Clasica)**: El estudio del movimiento de los cuerpos y las fuerzas que actúan sobre ellos (Cinemática, Leyes de Newton, Energía).
- 🌊 **[Ondas y Óptica](./Ondas_y_Optica)**: Oscilaciones, propagación de ondas, formación de imágenes, interferencia, difracción y sonido.
- ⚡ **[Electromagnetismo](./Electromagnetismo)**: El estudio de las cargas eléctricas, corrientes, campos magnéticos y las ondas electromagnéticas.
- 🔥 **[Termodinámica y Mecánica Estadística](./Termodinamica_y_Estadistica)**: El estudio del calor, el trabajo, la temperatura y el comportamiento estadístico de sistemas con muchas partículas.
- ⚛️ **[Mecánica Cuántica](./Mecanica_Cuantica)**: El estudio del comportamiento de la materia y la luz a escalas atómicas y subatómicas.
- 🚀 **[Relatividad](./Relatividad)**: La teoría de Einstein sobre el espacio, el tiempo y la gravedad (Relatividad Especial y General).
- ☢️ **[Física Nuclear y de Partículas](./Fisica_Nuclear_y_de_Particulas)**: Estructura del núcleo, decaimientos radiactivos, interacciones fundamentales y detectores.
- 💎 **[Materia Condensada](./Materia_Condensada)**: Cristales, bandas electrónicas, semiconductores, magnetismo y superconductividad.
- 🌌 **[Astrofísica y Cosmología](./Astrofisica_y_Cosmologia)**: El estudio de las estrellas, las galaxias y la evolución del universo en su conjunto.
- 📚 **[Recursos Útiles y Matemáticas](./Recursos_Generales)**: Matemáticas para física, computación científica, trabajo experimental, simuladores y literatura de apoyo.

## Cómo usar este repositorio

Cada carpeta contiene un archivo `README.md` que detalla los temas principales de esa rama y proporciona enlaces a:
- **Cursos online** (MIT OpenCourseWare, edX, Coursera, Khan Academy, etc.)
- **Libros de texto recomendados**
- **Artículos y vídeos explicativos**

*Si deseas contribuir con nuevos recursos o mejoras, siéntete libre de abrir un issue o un pull request.*
